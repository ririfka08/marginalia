# Marginalia - Book Shelf Project

*A quiet archive of stories worth remembering.*

Marginalia adalah arsip buku publik yang dikurasi secara personal. Pengunjung dapat menjelajahi koleksi, membaca ulasan singkat, dan menemukan rekomendasi buku berdasarkan tema atau minat — semuanya berdasarkan kurasi manusia, bukan algoritma.

## Target Pengguna

- Pembaca pemula yang ingin tahu harus mulai dari mana
- Pecinta sastra
- Mahasiswa
- Book lovers yang mencari rekomendasi yang lebih personal

## Fitur 

- **Browse Books** — jelajahi koleksi buku
- **Search** — cari buku berdasarkan judul, penulis, atau genre
- **Book Detail** — halaman detail tiap buku (sinopsis, rating, review singkat, tags)
- **Recommendation** — rekomendasi buku berdasarkan tema/minat
- **About** — tentang kurator dan Marginalia

> Admin panel sengaja tidak dibangun di v1. Data buku dikelola langsung melalui file terstruktur (JSON/Markdown) untuk menjaga scope tetap kecil dan fokus.

## Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** HTML, CSS, (JS jika diperlukan)
- **Design:** Figma
— catatan yang ditulis di pinggir halaman buku. Kadang berupa komentar, kadang tanda bahwa sebuah bagian layak diingat.*
